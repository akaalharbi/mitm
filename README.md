# Meet in the Middle Attack

## Introduction
This repository provides a versatile template for the meet-in-the-middle attack, adaptable to any block cipher, requiring only a *static library* that that will be acessed as described in `include/functions.h`. 
It's a modification of Wiener's approach, optimized for standard High Performance Computing infrastructure.
